# 南京师范大学附属中学黄山路小学

## 基本信息

| 项目 | 内容 |
|------|------|
| 别名 | 原：南京市建邺实验小学 |
| 性质 | 公办小学 |
| 建校时间 | 原建邺实验小学办学多年，2021年9月更名加入南师附中教育集团 |
| 地址 | 南京市建邺区黄山路122号（南京国际博览中心东南侧） |
| 教育集团 | 南师附中新城教育集团 |
| 现任校长 | 过静 |
| 副校长 | 丁维兵 |

## 历史沿革

1. **1950s-1990s**：经历多次更名（中胜办学点→中胜民办小学→中胜中心小学→沙洲中心小学）
2. **2007年8月**：更名为"南京市新城实验学校"
3. **2021年9月**：正式更名为"南京师范大学附属中学黄山路小学"，纳入南师附中新城教育集团

## 与南师附中新城小学的关系

> ⚠️ **重要区分：这是两所完全不同的学校**

| | 南师附中黄山路小学 | 南师附中新城小学 |
|---|---|---|
| 前身 | 建邺实验小学 | 新城实验学校 |
| 地址 | 黄山路122号 | 黄山路122号（另一校区） |
| 关系 | 同属南师附中新城教育集团 | 同属南师附中新城教育集团 |
| 参加派位 | ✅ 热点公办小学（2025年） | ❌ 从未参加电脑派位 |
| 施教区 | 江东中路以东·南河以西·河西大街以南·雨润大街以北 | 奥体大街与黄山路交汇处一带 |

> 搜狗百科记载南师附中新城小学"分黄山路和富春江两个校区"，但根据建邺区教育局2021年施教区划分，两校已分别独立，更名后的黄山路小学专指原建邺实验小学。

## 施教区范围

**大致范围**：江东中路以东、南河以西、河西大街以南、雨润大街以北

涵盖小区包括：海峡城、金地中心风华、绿地华侨城海珀滨江、正荣滨江紫阙等河西南片小区。

> 施教区以每年建邺区教育局官方文件为准，2026年具体范围待官方公布。

## 办学特色

- **校园活动丰富**：每年举办"哇蜗"采摘节（采摘石榴、桔子、柿子、柚子等）、艺术节、入学礼、开学典礼等
- **德育活动**：重视传统文化和爱国主义教育，开展过与新疆伊宁市第二十一中学的"民族团结美育交流"活动
- **科技/劳动教育**：组织科技创意挑战赛、劳动教育等活动
- **南师附中系资源**：共享南师附中教育集团的教学资源和师资培训体系

## 2025年电脑派位

- **类型**：热点公办小学
- **派位计划**：40人
- **派位范围**：建邺区
- **与张腾予关系**：张腾予对口致远乐山路分校，非该校施教区；但两校同在派位名单中
