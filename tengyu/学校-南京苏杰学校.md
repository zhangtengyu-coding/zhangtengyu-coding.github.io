# 南京苏杰学校

## 基本信息

| 项目 | 内容 |
|------|------|
| 学校性质 | 九年一贯制民办学校（小学+初中，无高中） |
| 创办时间 | 2000年9月 |
| 校长 | 苏平 |
| 校训 | **更快乐、更健康、更优秀** |
| 使命 | 用挚爱和智慧创造最适合学生的教育 |
| 办学目标 | 办中国最优秀的私立学校 |
| 业务主管 | 浦口区教育局 |
| 学费 | 约 17000元/学期 |
| 招生范围 | 面向南京全市招生（浦口区统一摇号） |
| 学校定位 | 学术最强，南外录取率全南京历史第一 |

## 教育理念

苏杰以"更快乐、更健康、更优秀"为校训，强调数学思维、研究能力、创新精神的培养。2019年创办初中部后，学术优势从小学延续至初中。

核心培养目标：
- 培养优秀的情商和品德
- 养成优秀的学习习惯和思维习惯
- 掌握科学的学习方法和思考方法
- 培养研究能力和创新能力

**背景：** 2000年创办，聘请哈佛、剑桥、北师大等国际教育专家研发课程，教师90%以上毕业于南大、南师大等名校。

|------|------|------|
| **浦口·老虎桥** | 浦口区江城大道29号 | 小学低段（1-4年级） |
| **浦口·科学城** | 浦口科学城（秋韵路附近） | 小学高段（5-6年级）+初中部，九年一贯制主阵地 |
| ~~鼓楼·青石村~~ | 鼓楼区青石村200号 | **已停用**，不再招生 |

> ⚠️ **2026.9是关键节点：** 河西南校区启用后，距离建邺区融侨中央花园仅约5km（地铁2号线2站），当前的距离障碍基本消失。

## 社团课程

各校区开设40+多元社团：乒乓球、篮球、足球、花样毽绳、健美操、排球、网球、武术、啦啦操、合唱、舞蹈、器乐等。

## 招生方式

- 全市电脑派位（摇号），在浦口区报名系统报名
- 6月10-11日填报志愿，6月24日公布派位结果
- 名额未单独公布（规模小，每年约60-100人）

## 对张腾予的意义

- **学术最强**：初中部中考均分南京第一，四星率100%，南外率历史最高
- **2026.9后距离问题消失**：河西南校区启用，地铁2号线2站即到
- **最大障碍**：当前浦口区摇号，通勤成本高
- **建议策略**：长期追踪，2026.9后可能成为首选
