# 南京青奥村外国语小学

## 基本信息

| 项目 | 内容 |
|------|------|
| 学校性质 | 民办（非营利性，南外系） |
| 地址 | 建邺区青奥南路10号 |
| 创办 | 2016年 |
| 合作方 | 南京外国语学校 + 建邺区政府 |
| 学费 | 17,000元/学期（约3.4万/年） |
| 班额 | 6个班，每班约27人 |
| 摇号名额（2025） | **228人**（↑84，扩招幅度最大） |

## 教育理念

- 目标：**培养有中国灵魂、世界胸怀的现代儿童**
- 愿景：**办一所让儿童更温暖、更开放、更自信成长的美好学校**
- 特色：以适切儿童的教育智慧与关怀，促进学生身心健康成长、德智体美劳全面发展

**背景：** 南外+建邺区政府合作办学，2016年创办。南外仙林优秀师资曾直接进驻，校长为原芳草园/莫愁新寓校长蒋丽云，师资阵容强大。

## 英语特色

- AB分班分层教学（南外标准）
- 南外原版教材（朗文、典范、感知世界、牛津）
- 沉浸式外语环境
- 各种英语竞赛、演讲、广播、英语节庆

> 青奥村定位：南外系第三所（南外仙林→南外→青奥村），外语优势突出。

## 升学通道（2024年重大突破）

2024年11月，青奥村外小与南京河西外国语学校签约"携手共进、贯通培养"：

- 每年设立 **1-2个直升班**，可直接升入**南外河西初中"数学英才班"**
- 这是**建邺区首个小学→初中直升通道**
- 数学拔尖创新人才培养：小学→南外河西初中→南外高中→北大数学英才班
