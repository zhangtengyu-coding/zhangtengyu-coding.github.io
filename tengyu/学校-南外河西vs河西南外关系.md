# 南外系学校关系梳理

## 南外家族总览

| 学校 | 性质 | 与南外本部关系 | 备注 |
|------|------|--------------|------|

## 关键区分：南京外国语学校河西初级中学 vs 南京河西外国语学校

**这是两所完全不同的学校！名字很像容易混淆：**

| | 南京外国语学校河西初级中学（南外河西） | 南京河西外国语学校（河西南外） |
|---|---|---|
| **性质** | 公办 | 民办 |
| **学段** | 初中为主（2012年创办） | **十二年一贯制**（小学到高中，2016年创办） |
| **管理** | 南外本部直接派遣 | 南外输出管理团队 |
| **地址** | 太清路 | 梦都大街60号 |
| **招生** | 学区直升+派位 | 全市招生，电脑派位 |

## 对张腾予的影响

- **南外本部**：只有初中，幼升小阶段不适用
- **南外河西二附小（公办小学）**：✅ 可以参加电脑派位
- **南京河西外国语学校（民办）**：✅ 可以参加电脑派位，面向全市
