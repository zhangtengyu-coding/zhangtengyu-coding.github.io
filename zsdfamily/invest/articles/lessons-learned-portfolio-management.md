> 这些不是规则，是思考方式。

---

## 一、预测报告 vs 持仓管理：必须严格分离

### 问题
预测报告里塞满了持仓成本、盈亏金额、止损止盈建议——把两种完全不同性质的内容混在一起，导致：
- 预测报告失去了客观性（被持仓信息污染）
- 持仓决策变得机械（被预测数据误导）

### 原则

**预测报告 = 纯技术分析，面向任何人，不带持仓信息**

- 只输出：技术信号、支撑压力位、趋势方向
- 不输出：买卖建议、止损止盈点位
- 结语只能说"偏强/偏弱/震荡"，不能说"建议买入/卖出"
- 任何人都可以读，不受特定持仓影响

**持仓管理 = 基于预测 + 真实持仓处境，具体问题具体分析**

- 读取：预测报告 + 持仓成本数据
- 输出：针对性操盘建议
- 每一个决策都是推理出来的，不是套公式

## 二、不教条主义：止损不是机械的

### 问题
"跌5%就止损"、"RSI>70就减仓"——这种规则听起来清晰，实际上害死人。

### 原则

**止损的本质是"承认错误"，不是"逃跑"。**

| 常见机械思维 | 正确思维方式 |
|-------------|------------|
| 跌5%就止损 | 看支撑位有没有跌破，支撑没破就是正常波动 |
| RSI>70就减仓 | RSI只是参考，要看均线是否发散、是否放量 |
| 突然暴跌>5%就割 | 先分析原因：大盘崩？黑天鹅？标的自身问题？处理完全不同 |
| 跌到成本线必须走 | 先问：当初买的逻辑还成立吗？成立就等，不成立才认错 |

**止损没有固定比例，关键是看"买入逻辑有没有被破坏"。**

## 三、不一刀切止盈：让利润奔跑还是落袋

### 问题
"涨15%必须止盈"——卖早了踏空，卖晚了利润回吐。

### 原则

**止盈看市场，不看固定比例。**

- 均线多头排列 + 放量上涨 → 趋势强，让利润跑一会儿
- 估值已经很贵 + 市场情绪亢奋 → 随时准备走
- 浮盈很大但标的逻辑很强 → 分批止盈，留底仓

**落袋为安和让利润奔跑都是对的，区别在于市场环境和你对标的的信心。**

## 四、被套了怎么办：没有标准答案

### 问题
被套之后只能"割肉"或"躺平"——非此即彼的思路太单一。

### 思考框架

拿到一个被套的持仓，先问自己：

```
1. 当初为什么买？买入逻辑是什么？
2. 这个逻辑现在还成立吗？
3. 标的本身变坏了吗？还是只是随大盘波动？
4. 套多久了？套多深？
5. 如果空仓，现在这个价格我会不会想买？
```

### 可选的应对方式（需要具体分析，不是每种都适用）

| 方式 | 适用场景 | 风险 |
|------|---------|------|
| 做T降成本 | 当天有足够波动空间，且你有把握 | 做反了成本更高 |
| 躺平等待 | 基本面没坏，市场会纠偏 | 熊市可能跌更深 |
| 割肉认赔 | 买入逻辑已证伪，承认错误 | 永远不卖在最低 |
| 加仓稀释 | 极确定的机会，不常出现 | 大多数时候是火坑 |
| 换仓 | 发现明显更好的标的 | 换来换去可能两边挨打 |

**没有一个选项是绝对正确的。每一个选择都要结合具体情况来判断。**

## 五、逢低加仓：慎之又慎

### 问题
被套之后补仓拉低成本，看起来很合理，实际上大多数时候是"接飞刀"。

### 原则

**加仓的前提：极确定的机会，而不是"觉得跌够了"。**

问自己：
- 我怎么知道这里就是底？
- 大盘是否已经企稳？
- 标的的基本面有没有根本性变化？
- 如果继续跌，我还能承受吗？

**大多数"逢低加仓"最后变成了"越套越深"。** 克制这个冲动。

## 六、做T：工具不是万能

### 问题
被套了做T降成本——听起来很聪明，做起来极难。

### 原则

**做T需要两个条件：**
1. 当天有足够的波动空间（至少1%以上）
2. 你对短时高低有判断能力

**做T做反了，成本反而更高。** 大多数人没有这个能力，不如果不动。

---

## 七、操盘建议要逐层披露

### 问题
一口气给出结论，不解释推理过程——用户看不到判断依据。

### 原则

**任务要分层拆解，逐步披露：**

```
第一层：持仓现状（成本、盈亏、份额）
↓
第二层：持仓处境分析（为什么买、逻辑还成立吗）
↓
第三层：技术背景摘要（从预测报告提取关键信息）
↓
第四层：综合判断（主要矛盾 + 我的看法）
↓
第五层：操作建议（买/卖/不动 + 理由 + 风险 + 跟踪事项）
```

每一层都要有实质内容，不是走过场。

## 八、市场不是考试

### 最重要的原则

**市场没有标准答案。没有哪个规则是必须遵守的。**

- 止损不是必须按固定比例
- 止盈不是到了某个点就必须卖
- 被套了不是只能割肉或躺平
- 大盘暴跌不一定要加仓，大盘暴涨不一定要卖

**规则是参考，是思考的起点，不是思考的终点。**

每一次决策，都要问自己：
- 我为什么这么做？
- 支持这个决策的理由是什么？
- 如果市场反向走，我的应对是什么？

## 九、具体问题具体分析的具体例子

### 场景1：浅套（<5%）

**不机械思维**：套了3%，要不要止损？

**正确思路**：
- 成本线附近有没有强支撑？有支撑就等
- 大盘是在牛市还是熊市？牛市里套3%不值得焦虑
- 买的逻辑还成立吗？成立就当正常波动

### 场景2：深套（>10%）

**不机械思维**：套了15%，必须割肉

**正确思路**：
- 当初买的逻辑是什么？这个逻辑现在还成立吗？
- 标的本身变坏了吗？还是只是随大盘跌？
- 如果逻辑坏了，认错换仓；如果逻辑没坏，等

### 场景3：微利（3%以内）

**不机械思维**：小赚就要落袋

**正确思路**：
- 市场情绪如何？强趋势可以多拿一会儿
- 均线多头排列吗？是的话让利润跑

### 场景4：大赚（>10%）

**不机械思维**：涨了15%必须全部止盈

**正确思路**：
- 趋势还在吗？强趋势不轻易下车
- 估值贵不贵？已经很贵了就开始分批走
- 留底仓：减了之后留一部分，既落袋又不错过

## 十、时间窗口与持仓管理的关系

**定时任务是定时任务，持仓管理是持仓管理。**

- 定时任务的决策窗口是14:00-14:55（判断当天是否操作）
- 但持仓管理本身可以随时调用，不需要等定时任务
- 用户问"帮我分析一下008114"——随时可以分析
- 定时任务只是确保在最佳时机自动触发，不是限制分析的时间
