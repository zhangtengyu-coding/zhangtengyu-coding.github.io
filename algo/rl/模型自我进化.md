# 模型自我进化（Self-Evolution）

## 来源

MiniMax M2.7（2026-03-18）技术博客。

## 什么是模型自我进化

传统 RL 训练是由人类工程师设计 reward、设计环境、设计脚手架，模型被动接受训练。

自我进化是指：**模型能够主动参与自身的迭代过程**。

M2.7 是第一个实现深度自我进化的模型：
- 自行构建复杂 Agent Harness（测试框架）
- 驱动强化学习过程
- 自主迭代优化 Agent 架构、Skills/MCP 实现和记忆机制

## 关键区别

| 传统 RL | 自我进化 |
|---------|---------|
| 人类设计 harness | 模型自行构建 harness |
| 人类定义 reward | 模型参与 reward 设计 |
| 人类选择训练数据 | 模型驱动数据生成 |
