# 评测基准索引

## 编程能力

### SWE-Bench 系列
| 基准 | 说明 | M2.5 成绩 |
|------|------|-----------|
| SWE-Bench Verified | 真实 GitHub issue 修复，通过率 80.2% | 80.2% |
| Multi-SWE-Bench | 多语言编程任务 | 51.3%（全球第一） |
| SWE Multilingual | 跨语言编程 | 76.5% |
| SWE-Pro | 端到端项目交付 | 56.22%（M2.7） |
| VIBE-Pro | 端到端完整项目交付 | 55.6%（M2.7） |
| Terminal Bench 2 | 复杂工程系统深层理解 | 57.0%（M2.7） |

### 脚手架泛化
在 Droid、OpenCode、Cline、Cursor 等不同编程脚手架上测试 SWE-Bench Verified：
- M2.5 Droid: 79.7（超过 Opus 4.6 的 78.9）
- M2.5 OpenCode: 76.1（超过 Opus 4.6 的 75.9）

## 搜索能力

### BrowseComp
真实浏览器搜索任务，考察复杂信息检索能力：M2.5 达到 76.3%。

### RISE（Realistic Interactive Search Evaluation）
MiniMax 自建的真实专家级搜索评估——模拟人类专家在专业网页内的深度探索，而非简单搜索引擎查询。

### Wide Search
多范围搜索任务综合评估。

## Agent 能力

### Toolathon
工具调用正确率：M2.7 达到 46.3%（全球第一梯队）。

### OpenClaw
Agent 综合评测：接近 Sonnet 4.6。

## 专业办公

### GDPval-MM（GDPval-AA）
两两对比评估模型交付质量和轨迹专业性：
- M2.7 GDPval-AA ELO: 1500（仅次于 Opus 4.6 / Sonnet 4.6 / GPT5.4）
- M2.5 GDPval-MM 胜率: 59.0%

## 机器学习工程

### MLE Bench Lite
22 个机器学习任务（Kaggle 风格）：M2.7 取得 9 金 / 5 银 / 1 铜。
